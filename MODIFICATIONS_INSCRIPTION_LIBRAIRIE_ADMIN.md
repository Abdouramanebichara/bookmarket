# Modifications inscription librairie et validation administrateur

## Inscription librairie

La création d'un compte librairie passe maintenant par une demande de validation envoyée à l'administrateur.

La librairie doit obligatoirement fournir :

- nom du responsable ;
- nom de la librairie ;
- email professionnel ;
- téléphone ;
- adresse complète ;
- ville ;
- pays ;
- latitude GPS ;
- longitude GPS ;
- numéro RCCM ;
- NIU / carte de contribuable ;
- statut juridique ;
- mot de passe.

## Documents obligatoires transmis à l'administrateur

- RCCM ;
- carte de contribuable / NIU ;
- attestation de création ;
- CNI du responsable ;
- plan de localisation ;
- contrat de bail ou preuve d'adresse.

## Document optionnel

- patente ou justificatif fiscal.

## Administration

L'administrateur peut maintenant :

- consulter les demandes d'inscription librairie ;
- voir les coordonnées GPS ;
- voir les informations légales ;
- ouvrir les documents transmis ;
- télécharger les documents transmis ;
- accepter la création de la librairie ;
- refuser la création de la librairie.

Après acceptation, la librairie devient un utilisateur de la plateforme et peut se connecter avec l'email et le mot de passe renseignés lors de la demande.

## Logique connexion / création

- un client créé apparaît dans la gestion des utilisateurs côté administrateur ;
- une librairie en attente ne peut pas se connecter ;
- une librairie refusée ne peut pas se connecter ;
- une librairie acceptée peut se connecter ;
- un compte désactivé par l'administrateur ne peut pas se connecter.
